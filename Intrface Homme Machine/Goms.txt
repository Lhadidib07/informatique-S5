GOMS : model de discription du comportement humain 

( Goal, Operation , Methode , Selection )

goal : le bute , 

operation : les action elementaire 

methodes : les diffrentes methodes bash ndirha ( clavier ou souris )  

slection : choix de la methodes en cas de conflit ( soris indspo choisie le clavier )  

technique pour predire le temps d'excution d'une tache routiniére par un utilisateur sans erreur 

**** avantages : 
 	
	- methodes de conception compatibles avec celles des informaticiens 
	
	- mesurer le comrtoment de l'utilisateur 
	- technique simple 
	- pour comparer et predire les preformances 

**** inconveniant  : 

	- ne peut pas etre utlisé dans une interfaces de reconnaissance vocal ou gestuelle 
	- impossible de prendre en compte les erreurs 
	- il n'est pas precis
